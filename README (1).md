# 💕 Quiz Spesial Buat Kamu

Quiz interaktif yang indah dan romantis dengan animasi yang menarik.

## 🎯 Fitur

- ✨ Quiz dengan 10 pertanyaan
- 🎨 Desain responsive dan animasi smooth
- 💝 Confetti dan falling hearts animation
- 🎵 Sound effects untuk setiap aksi
- 📱 Mobile-friendly design

## 🚀 Cara Menggunakan

1. Buka file `index.html` di browser Anda
2. Klik tombol "Mulai Quiz"
3. Jawab 10 pertanyaan dengan jujur
4. Lihat hasil dan jawab pertanyaan terakhir
5. Nikmati animasi perayaan! 🎉

## 📋 Struktur File


## 🎨 Warna & Tema

- **Primary**: Pink (#ff4d6d, #ff2d5a)
- **Secondary**: Gradient pink cantik

## 💻 Teknologi

- HTML5
- CSS3 (Animasi, Gradient, Media Query)
- JavaScript (Vanilla - tanpa library)

## 📝 Catatan

- Semua pertanyaan bisa disesuaikan di bagian `quizData`
- Animasi dan efek suara bisa dikustomisasi di CSS
- Responsif sempurna untuk semua ukuran layar
